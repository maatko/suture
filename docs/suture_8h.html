◆ su_detour()

enum su_error su_detour ( struct su_env * env,
const char * class_name,
const char * method_name,
const char * method_signature,
jmethodID * original_method,
void * function )

Registers a detour on a method identified by class name, method name, and signature.

Resolves the target method via JNI and schedules a bytecode detour to redirect execution to function. The original method remains accessible via original_method.

Parameters
envInitialised Suture environment. Must not be NULL.
class_nameFully qualified class name (e.g. "java/lang/Object"). Must not be NULL.
method_nameName of the method to detour. Must not be NULL.
method_signatureJNI method descriptor (e.g. "(I)V"). Must not be NULL.
original_methodOut parameter receiving the original method ID. May be NULL.
functionDetour function pointer to redirect execution to. Must not be NULL.
Returns
SU_OK on success.
SU_MISSING_REQUIRED_PARAMETERS if any required parameter is NULL.
SU_DETOUR_INVALID_TARGET if the class or method could not be resolved.
SU_MEMORY_ALLOCATION_FAILURE if the hook could not be registered.

◆ su_dispose()

enum su_error su_dispose ( struct su_env * env)

Detaches from JVMTI and releases all resources owned by the environment.

Frees all hooks, targets, and JVMTI state. Does not free env itself. After this call, env must not be used without re-initialising via su_init.

Parameters
envEnvironment to dispose. Must not be NULL.
Returns
SU_OK on success.
SU_MISSING_REQUIRED_PARAMETERS if env is NULL.
SU_JVM_JVMTI_DETACH_FAILURE if the JVMTI environment could not be cleanly detached.

◆ su_init()

enum su_error su_init ( struct su_env * env)

Initialises a Suture environment, attaching to the running JVM and acquiring JVMTI capabilities.

Must be called before any other Suture function. Locates the active JVM, attaches a JVMTI environment, and registers the class file load hook.

Parameters
envEnvironment to initialise. Must not be NULL.
Returns
SU_OK on success.
SU_MISSING_REQUIRED_PARAMETERS if env is NULL.
SU_JVM_NO_VIRTUAL_MACHINES if no running JVM could be found.
SU_JVM_JVMTI_ATTACH_FAILURE if the JVMTI environment could not be attached.
SU_JVM_CAPABILITIES_FAILURE if required JVMTI capabilities could not be acquired.

enum su_error su_mdetour ( struct su_env * env,
jmethodID method,
jmethodID * original_method,
void * function )

Registers a detour on a method identified by an existing jmethodID.

Equivalent to su_detour but takes a pre-resolved jmethodID directly, bypassing JNI class and method lookup.

Parameters
envInitialised Suture environment. Must not be NULL.
methodResolved method ID to detour. Must not be NULL.
original_methodOut parameter receiving the original method ID. May be NULL.
functionDetour function pointer to redirect execution to. Must not be NULL.
Returns
SU_OK on success.
SU_MISSING_REQUIRED_PARAMETERS if env or function is NULL.
SU_DETOUR_INVALID_TARGET if method could not be resolved to a class.
SU_MEMORY_ALLOCATION_FAILURE if the hook could not be registered.

◆ su_transform()

enum su_error su_transform ( const struct su_env * env)

Applies all registered detours by triggering a JVMTI class retransformation.

Retransforms every class in env->targets, causing the JVMTI class file load hook to rewrite their bytecode with the registered detours applied.

Parameters
envInitialised Suture environment with at least one registered hook. Must not be NULL.
Returns
SU_OK on success.
SU_MISSING_REQUIRED_PARAMETERS if env is NULL.
SU_JVM_RETRANSFORM_FAILURE if JVMTI retransformation fails.
