enum su_error su_flag_patchb ( const char * name,
bool * original,
bool value )

Patches a JVM boolean flag by name.

Walks the HotSpot VM structs to locate the named flag in the JVM flag table and overwrites its value. Supports both legacy (Flag) and modern (JVMFlag) layouts.

Parameters
nameName of the JVM flag to patch. Must not be NULL.
originalIf non-NULL, receives the flag's original value before patching.
valueNew boolean value to write to the flag.
Returns
SU_OK on success.
SU_MISSING_REQUIRED_PARAMETERS if name is NULL.
SU_DETOUR_NOT_SUPPORTED if the flag table or target flag could not be located.
