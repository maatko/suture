extern

JVMTI class file load hook that intercepts and retransforms loaded classes.

Registered with JVMTI's ClassFileLoadHook event. Invoked for every class load or retransformation, allowing Suture to rewrite bytecode before the JVM resolves it.

Parameters
jvmtiJVMTI environment.
jniJNI environment.
class_being_redefinedClass being redefined, or NULL on initial load.
loaderClass loader, or NULL for the bootstrap loader.
nameInternal class name (e.g. "java/lang/Object").
protection_domainProtection domain of the class, or NULL.
class_data_lenLength in bytes of class_data.
class_dataOriginal class file bytes.
new_class_data_lenOut parameter for the length of the rewritten class data.
new_class_dataOut parameter for the rewritten class file bytes.
