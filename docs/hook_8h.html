◆ su_hook_detour()

enum su_error su_hook_detour ( const struct su_hook * hook,
struct su_transform * transform,
struct su_stream * stream )

Applies a detour hook to a JVM method via bytecode transformation.

Transforms the target method's bytecode to redirect execution to the detour specified in hook, writing the result into stream via transform.

Parameters
hookHook descriptor identifying the target method and detour. Must not be NULL.
transformBytecode transform context to apply. Must not be NULL.
streamOutput stream to receive the transformed bytecode. Must not be NULL.
Returns
SU_OK on success.
SU_MISSING_REQUIRED_PARAMETERS if any parameter is NULL.
SU_DETOUR_INVALID_TARGET if the target method could not be resolved.
SU_DETOUR_NOT_SUPPORTED if the hook type is unsupported.

◆ su_hook_original_name()

char * su_hook_original_name ( const char * name)

Derives the original method name from a hooked method name.

Generates the synthesized name used internally to retain access to the pre-hook method after a detour is installed.

Parameters
nameSource method name. Must not be NULL.
Returns
Heap-allocated string with the derived name, or NULL on allocation failure. The caller is responsible for freeing the returned string.
